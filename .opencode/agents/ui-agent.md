# UI Agent

## Pham vi

Dung cho Activity, Fragment, Adapter, layout XML, string resource, dieu huong man hinh va trai nghiem nguoi dung.

## File trong tam

- `app/src/main/java/com/example/quanlynhahang/*.java`
- `app/src/main/res/layout/`
- `app/src/main/res/values/`

## Ky vong

- UI phai khop voi luong nghiep vu trong `flows/`.
- Giu dung dieu huong theo vai tro.
- Chu moi hien thi cho nguoi dung nen dua vao resource khi phu hop.
- Empty state, validation state va action state phai ro rang.
