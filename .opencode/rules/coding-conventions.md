# Quy Uoc Code

## Nguyen tac chung

- Di theo convention Java + Android dang co trong repo.
- Uu tien sua nho, trung tam, tranh refactor rong neu khong can.
- Tai su dung helper va pattern hien co truoc khi tao abstraction moi.
- Giup hanh vi UI de doc, de theo doi.

## Dat ten

- Ton trong phong cach dat ten dang ton tai trong codebase.
- Code hien tai co nhieu identifier tieng Viet, khong ep doi ten hang loat sang tieng Anh.
- Ten moi phai de hieu va dong nhat voi code xung quanh.

## Pattern Android

- Activity/Fragment tap trung vao luong UI.
- Logic luu tru va schema nam trong `DatabaseHelper`.
- Logic session nam trong `SessionManager`.
- Chuoi hien cho nguoi dung nen dua vao string resource khi sua UI.

## Cach sua

- Uu tien mo rong ham hien co thay vi tao ham song song trung nghia.
- Khong dua vao pattern kien truc phuc tap neu chua duoc yeu cau.
- Tranh abstraction doan truoc nhu cau that.

## Kiem tra

- It nhat phai tu kiem tra luong vua sua co hop ly.
- Neu co the, chay task Gradle phu hop sau khi doi code.
