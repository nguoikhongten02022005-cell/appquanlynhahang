# Skill: Thay Doi DB An Toan

## Muc dich

Dung khi them cot, doi logic status, doi insert/update, hoac chinh seed data.

## Quy tac

- Migration schema phai idempotent.
- Cot moi nen co default an toan.
- Phai giu kha nang doc du lieu lich su.
- Tranh migration pha du lieu local neu nguoi dung chua yeu cau ro.

## Cac buoc

1. Xac dinh bang va ham bi anh huong trong `DatabaseHelper`.
2. Cap nhat logic dam bao schema.
3. Cap nhat tat ca noi doc va ghi du lieu lien quan.
4. Cap nhat UI neu du lieu moi lam thay doi cach hien thi.
5. Verify lai seed/default data.

## Diem de gay loi

- May cu chua co cot moi
- Lech enum/status giua DB va UI
- Vo counter/thong ke sau khi doi schema
- Session bi anh huong gian tiep boi thay doi du lieu nguoi dung
