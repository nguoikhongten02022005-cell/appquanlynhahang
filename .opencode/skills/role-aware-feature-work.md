# Skill: Tinh Nang Co Xet Vai Tro

## Muc dich

Dung khi yeu cau dong vao dang nhap, dang ky, tai khoan, cong cu nhan vien, cong cu admin, hoac bat ky luong redirect nao.

## Quy tac

- Xac dinh vai tro dich truoc.
- Giu dung viec chan truy cap trai phep.
- Giu dung diem den sau dang nhap.
- Cap nhat ca phan UI lan gia dinh session ben duoi.

## File thuong bi anh huong

- `LoginActivity`
- `RegisterActivity`
- `MainActivity`
- `EmployeeActivity`
- `AdminActivity`
- `SessionManager`

## Verify

- Luong nguoi chua dang nhap phai dung
- Luong khach hang phai dung
- Luong nhan vien phai dung
- Luong admin phai dung
- Luong session loi phai fail-safe
